##Документация проекта
Описание базового синтаксиса записи и форматирования
Проект использует следующие технологии и синтаксис:

Python: Основной язык программирования для написания тестов.
Pytest: Фреймворк для написания и запуска тестов.

##Форматирование кода
Код форматируется в соответствии с PEP 8 (стиль написания кода на Python).
Используются docstrings для документирования методов и функций.


##Инструкция по получению токена авторизации.
Перейдите на сайт https://developers.sber.ru/.
Зарегистрируйте аакаунт, используя email или войдите с помощью СберID.
Получите Authorization key.
Войдите в личный кабинет и скопируйте Authorization key.
Создайте файл Authorization_key.txt в корневой директории проекта и вставьте в него значение Authorization key.
Создайте файл token.txt и оставьте его пустым.

##Установите необходимые зависимости:
pip install -r requirements.txt

Запустите тесты с помощью testing конфигурации в vs code или в терминале с помощью команды pytest test_API_giga.py -v -s.
